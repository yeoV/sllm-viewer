# Viewer for LLM

- LLM 추론 / 성능 비교를 위한 웹만들기

### Requirements

- local python version : 3.11.4
- venv

### Install

- Step1. Create virtual env & activate

```
# create python virtual environments
python -m venv env
source env/bin/activate
```

- Step2. Install poetry

```
pip install poetry
```

- Step3. Install poetry env

```
poetry install
```
